# B2C E-Commerce Project

A full-stack e-commerce application using the **MERN Stack**.

## 📁 Project Structure
- **/frontend**: Built with React.js (Client-side).
- **/backend**: Built with Node.js & Express (Server-side API).

## 🛠 Tech Stack
- **Frontend**: React, Axios, React Router.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Mongoose).

## ⚙️ How to Start
1. For Backend: `cd backend && npm install && npm start`
2. For Frontend: `cd frontend && npm install && npm start`
